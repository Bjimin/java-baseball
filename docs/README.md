## 구현할 기능 목록

- 랜덤 수 추출
  - 1~9까지 서로 다른 임의의 수 3개 (v)  


- 사용자 입력 받기
  - 입력 받기 (v) 
  - 잘못된 값 입력 판단 (v)  


- 컴퓨터 수와 사용자 입력 값 비교
  - 스트라이크 카운트 (v)
  - 볼 카운트 (v)
  - 낫싱 판단 (v)
  - 계산 결과 출력 (v)  
  

- 게임 종료 및 재시작
  - 게임 종료 메서드 (v)
  - 게임 재시작 메서드 (v)   
  

- 예외 처리
  - 게임 종료 및 제시작 값 잘못 입력 (v)
  - 중복된 수 입력 (v)
  - 정해진 범위의 숫자 외의 값 입력 (v)
  - 길거나 짧은 길이의 수 입력 (v)
